◆◆◆ 第一章 · 游戏愿景 ◆◆◆
────────────────────────────────────────

【一句话概括】
  你是坠入深渊的最后一位守墓人，在无尽怪潮中靠自动武器杀出生路，
  每一次死亡都让你变得更强，直到你足以斩杀深渊之主。

【核心体验三角】
  ┌─────────────┐
  │   生存压力    │ ← 前5分钟：活下来就是胜利
  │   ×          │
  │   构筑快感    │ ← 中期：组合武器 + 被动 = 你的专属Build
  │   ×          │
  │   割草爽感    │ ← 后期：满屏特效、万敌消融的视觉高潮
  └─────────────┘

【美术风格】
  2D暗黑手绘像素风 · 俯视角45°
  - 角色：16×16像素基础，放大渲染带描边
  - 地面：分层Tilemap + 动态阴影
  - 特效：粒子系统 + 全屏后处理（泛光 / 色差 / 屏震）
  - 色调：以深紫、暗红、幽绿为主，高光用金色/白色

【参考作品】
  吸血鬼幸存者 × 黑帝斯（叙事深度）× 以撒（随机性）× 土豆兄弟（组合性）

【单局流程】
  1. 选择角色 + 初始武器
  2. 进入随机地图，怪潮自动开始
  3. 30分钟倒计时，活下来即为胜利
  4. 每次升级 3~4 选 1，构建Build
  5. 每10分钟遭遇阶段
  6. 30分钟时面对最终BOSS
  7. 结算 → 获得灵魂碎片（局外货币）→ 永久强化

【操作方案】（单机）
  键盘：WASD 移动 / ESC 暂停 / 1~6 查看武器详情
  手柄：左摇杆移动 / 开始键暂停
  ※ 无需攻击键，所有武器全自动！

  属性名       初始值      说明
  ─────────   ────────   ─────────────────────────────
  生命值       100        归零即死，可通过回复/吸血恢复

  闪避率       0%         概率完全闪避一次攻击，上限60%
  每秒回血     0          每秒自动恢复的生命，上限5
  暴击率       5%         暴击时造成150%伤害
  暴击伤害     150%       暴击时的伤害倍率

  伤害加成     0%         全局伤害增幅

【3.2 可选角色】（初始解锁1个，其余通过成就解锁）

  ◇ 守墓人·阿什 [初始角色]
    - 专属被动「墓地之力」：击杀敌人有5%概率掉落小型回复
    - 初始武器：魔法飞弹
    - 属性偏向：均衡型

  ◇ 暗影刺客·莉拉 [成就解锁：10分钟内击杀2000敌人]
    - 专属被动「暗影步」：闪避成功后0.5秒内移速+80%
    - 初始武器：回旋镖
    - 属性偏向：高速度、低生命

  ◇ 铁壁骑士·加隆 [成就解锁：单局护甲达到30]
    - 专属被动「不屈意志」：血量低于30%时护甲翻倍
    - 初始武器：白骨之盾
    - 属性偏向：高生命、高护甲、低速度

  ◇ 炼金术士·菲奥 [成就解锁：单局触发5次武器进化]
    - 专属被动「万物融合」：武器进化所需等级-2
    - 初始武器：烈焰之球
    - 属性偏向：高伤害、低闪避

  ◇ 裂隙行者·虚无 [成就解锁：通关最终BOSS]
    - 专属被动「次元漫步」：每30秒自动闪现到安全位置
    - 初始武器：虚空之眼
    - 属性偏向：特殊（属性随机波动）

  ◇ ???·死神 [隐藏角色：30分钟内无伤通关]
    - 专属被动「收割」：击杀敌人后1秒内攻击+100%
    - 初始武器：死神镰刀（隐藏武器）
    - 属性偏向：极端（攻击极高、生命极低50）

【4.2 武器升级路径】

  每个武器 1~8 级，每次升级可选一种强化词条：

  通用词条：
    · 伤害 +10%
    · 冷却 -8%（叠加计算）
    · 范围 +12%
    · 持续时间 +0.5秒
    · 穿透 +1（投射物类）
    · 弹射 +1（追踪/远程类）
    · 数量 +1（投射物/召唤物/地刺）
    · 击退力 +20%

  稀有词条（5级后随机出现）：
    · 附加灼烧：命中后3秒内额外造成总伤害20%的火焰伤害
    · 附加冰冻：5%概率冻结敌人1秒
    · 附加吸血：造成伤害的5%转化为生命恢复
    · 连锁闪电：命中后15%概率链式弹跳到旁边1个敌人
    · 弱点标记：被命中的敌人受到全部伤害+8%，持续3秒

  · 暗影分身 + 虚空之眼 → 「深渊凝视军团」：分身各带一只虚空之眼

◆◆◆ 第六章 · 经验与升级系统 ◆◆◆

【6.1 经验宝石】

  击杀敌人掉落经验宝石，根据敌人强度：
    · 普通杂兵 → 蓝色小宝石 (1 EXP)
    · 强化杂兵 → 绿色中宝石 (3 EXP)
    · 精英怪   → 黄色大宝石 (10 EXP)
    · BOSS     → 红色巨型宝石 (50 EXP)

  宝石在拾取范围内自动飞向玩家。
  地面放置超过45秒的宝石开始闪烁，60秒后消失。

【6.3 升级选项机制】

  每次升级时，游戏暂停，显示 3 个随机选项（可通过被动提升到4个）。

【10.1 伤害公式】

  最终伤害 = 基础伤害
             × (1 + 全局伤害加成)
             × (1 + 武器特定加成)
             × 暴击倍率 [未暴击=1.0, 暴击=暴击伤害%]
             × 克制系数 [通常=1.0, 克制=1.3, 被克=0.7]
             - 目标护甲 [固定减伤]

  ※ 最小伤害保底为1，不会出现0伤害

【10.2 防御公式】

  实际受到伤害 = 敌人伤害 × (1 - 闪避判定) - 护甲值
  其中闪避判定：随机数 < 闪避率 则本次伤害为0

【10.3 冷却计算】

  实际冷却 = 基础冷却 × (1 - 全局冷却缩减) × (1 - 武器冷却缩减)
  ※ 最低冷却不低于基础的30%，防止无限攻击

【10.4 经验倍率】

  实际经验 = 基础经验 × (1 + 经验加成) × 地图倍率 × 难度倍率

【13.1 天气/事件系统】

  每局随机触发 2~4 次全局事件（持续30~60秒）：

  ·「血月升起」
     效果：全屏变为红色调，敌人伤害+30%但经验掉落+50%
     策略：高风险高回报时段

  ·「黄金时刻」
     效果：30秒内所有掉落的经验宝石价值×3
     策略：尽量在此期间多击杀

  ·「迷雾弥漫」
     效果：视野缩小至200像素范围，但敌人移速-30%
     策略：小心走位，避免被偷袭

  ·「深渊潮汐」
     效果：地图边缘向内涌入暗影，安全区缩小50%持续45秒
     策略：向中心聚集，考验生存能力

  ·「圣光庇护」
     效果：玩家周围100像素光环，光环内回血+3/秒
     策略：短暂喘息机会

  ·「时间加速」
     效果：一切速度×1.5（含敌人和玩家），持续30秒
     策略：手速和反应考验

【13.2 元素克制系统】

  武器和敌人有隐藏元素属性：
    火 > 冰 > 雷 > 火 (三角克制)
    暗 ↔ 光 (互相克制)
    无属性不受克制

  克制倍率：伤害×1.3
  被克倍率：伤害×0.7

  ※ 不强制，但合理搭配武器应对特定地图/敌人可获得优势

【13.3 连击奖励系统】

  连续击杀不间断（2秒内下一次击杀）触发连击：

  连击数     奖励
  ────────  ────────────────────────────
  50        经验获取+10%（本次连击期间）
  100       移动速度+10%（本次连击期间）
  200       全局伤害+15%（本次连击期间）
  500       全屏闪光 + 10秒无敌
  1000      成就「千人斩」+ 全属性暴涨30% 10秒

  连击中断时所有加成消失。

【13.4 「深渊回响」日记系统】

  玩家在游戏中可收集散落的日记碎片（稀有掉落），
  逐步揭示世界观和故事：

  · 守墓人为何坠入深渊？
  · 骷髅王生前是谁？
  · 深渊之主的真实身份
  · 武器背后的古代传说
  · 各地图的历史

  日记碎片在局外永久保存，可在主菜单「图书馆」中阅读。
  集齐一个章节解锁对应的角色皮肤。

【13.5 每日挑战种子（单机版）】

  每天根据日期生成一个固定种子的挑战局：
    · 固定地图 + 固定初始武器 + 固定敌人序列
    · 完成后显示评分（基于存活时间、击杀数、Build效率）
    · 与自己的历史最佳比较
    · 每周一重置排行，保留个人最佳记录

◆◆◆ 第十五章 · 技术实现要点 ◆◆◆
────────────────────────────────────────

【15.1 性能优化（关键）】

  割草游戏同屏可能有数百上千个实体，性能是第一要务：

  · 空间分区：使用四叉树或网格分区管理碰撞检测
    - 屏幕分为 NxN 网格，只检测同格/相邻格内的碰撞
    - 目标：同屏1000+敌人稳60FPS

  · 对象池：
    - 敌人、子弹、粒子全部使用对象池
    - 避免频繁GC卡顿

  · 渲染优化：
    - 离屏敌人不绘制（仅更新逻辑）
    - 粒子效果使用精灵批处理
    - 低配模式：减少粒子数量/关闭后处理

  · LOD（细节层次）：
    - 远处敌人简化渲染（单色圆点）
    - 满屏时自动降低特效粒子数

【15.2 存档系统（单机）】

  · 局外数据（永久保存）：
    - 灵魂碎片数量
    - 永久升级进度
    - 解锁角色/地图/武器
    - 成就完成度
    - 图鉴收集进度
    - 统计数据（总击杀、总游戏时间等）

  · 局内数据（可选：存档中退）
    - 当前Build状态
    - 敌人分布快照
    - 游戏时间进度
    - ※ 中退存档为一次性，读取后删除（防止SL）

  · 存档格式：JSON + 简单校验和（防止简单篡改）
  · 存档位置：游戏目录下 /saves/profile.json

【15.3 配置系统】

  · 画面设置：
    - 分辨率：窗口化 / 全屏 / 无边框
    - 帧率上限：30/60/120/不限
    - 粒子数量：低/中/高
    - 屏震强度：0%/50%/100%
    - 泛光效果：开/关

  · 音频设置：
    - 主音量/音乐/音效 独立滑条

  · 游戏性设置：
    - 自动拾取范围显示：开/关
    - 伤害数字显示：开/关/仅暴击
    - 连击计数显示：开/关

【15.4 分布式服务器架构（创新特性）】

  游戏可以向玩家授权，当游戏在玩家电脑运行时，玩家电脑也会变成服务器的一部分。

  · 架构原理：
    - 玩家启动游戏客户端后，客户端可兼任微型服务节点
    - 经玩家明确授权后，利用空闲计算资源参与分布式网络
    - P2P 网络拓扑，自动发现和连接附近节点

  · 应用场景：
    - 多人匹配中继：减少中心服务器负载
    - 排行榜同步：分布式存储和同步全球排行
    - 资源 CDN 分发：玩家间互助下载游戏更新/资源包
    - 世界事件广播：实时共享全球游戏事件（如稀有怪物出现）
    - 云存档备份：分布式备份玩家存档，防止数据丢失

  · 安全措施：
    - 端到端加密通信（TLS/SSL）
    - 沙盒环境隔离，不访问玩家其他文件
    - 严格的带宽和 CPU 占用可调上限（默认 ≤5% 系统资源）
    - 仅转发验证过的游戏数据，拒绝恶意请求
    - 玩家可随时关闭授权，完全退出网络

  · 激励机制：
    - 贡献算力获得游戏内奖励：
      * 每小时在线并授权 → 10 灵魂碎片
      * 协助转发数据包 → 额外金币/材料
      * 成为稳定节点 → 限定装饰品/称号
    - 贡献排行榜，月度奖励最活跃贡献者

  · 离线可用性：
    - 即使中心服务器维护/下线，玩家网络也能自组织运行
    - 单机模式完全不依赖网络，离线可完整游玩
    - 社区自治：玩家可建立独立子网络（如局域网赛事）

  · 技术实现建议：
    - 使用 libp2p 或 WebRTC 数据通道建立 P2P 连接
    - DHT (分布式哈希表) 存储节点信息和排行榜数据
    - IPFS 协议用于资源分发
    - 区块链技术（可选）记录贡献值，防止作弊

  · 用户体验：
    - 首次启动时弹出授权说明，明确告知资源占用和收益
    - 设置界面可查看实时贡献统计（上传/下载流量、在线时长等）
    - 一键开关「成为节点」功能
    - 提供「性能模式」选项：仅玩 / 玩+轻度贡献 / 玩+全力贡献
